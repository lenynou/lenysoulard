Jeu de comabt sur Unity :

Etant joueur de jeux-vidéo avec une certaine attache pour les jeux de combats de type "plateformer" et étant en étude dans le domaine de l'informatique, je me suis demandé comment les jeux-vidéo était conçu. A partir de cette question, j'ai recherché les outils permettant de concevoir des jeux. J'ai donc décidé, après mon exploration dans le vaste monde du développement de jeux, d'utiliser le moteur de jeu Unity.

A l'aide de tutoriels et d'IA comme "ChatGPT", je me suis donc lancé dans le projet de réaliser un petit jeu de combat. Pour cela, j'ai donc décidé de réaliser une premiere version pour me familiariser avec le logiciel. Puis, je ferais une version plus terminé visuellement et avec un code plus propre. A l'heure actuelle je pense avoir fini la première version et je commencerai donc la version finale des que possible.
