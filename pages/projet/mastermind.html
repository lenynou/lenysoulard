Le MasterMind :

Le jeu du MasterMind est un jeu très connu qui se joue a 2 joueurs, un décodeur et un codeur. Le but pour le joueur décodeur est de trouver un code a 4 couleur en un mininmum de manche et le joueur codeur doit choisir un code et donné des indications sur les code proposé par le décodeur. Ce projet, qui est le premier que j'ai fais dans le cadre de mon BUT, avait pour but de réaliser une version numérique de ce jeu.

Etant donné que ce projet était le premier que nous réalisions avec mon binome, le code a été fait de façon procédurale et l'affichage graphique était une extension proposé. Bien que ce projet n'a pas été concu avec un code propre et de maniere simpliste, cela m'a montré les difficultés à travailler a plusieurs sur un projet informatique.
